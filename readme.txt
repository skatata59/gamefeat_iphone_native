-------------------
■GameFeat SDK
管理画面よりSDKをダウンロードし、
プロジェクトに取り込んでください。

-------------------
■使用方法
【ViewController.m　12行目】
#define GAMEFEAT_MEDIA_ID @"メディアID"
を管理画面より取得したメディアIDに置き換えてください。

-------------------
■URLスキームの設定
相互送客機能を使用するために、アプリに任意のURLスキームを設定してください。
URLを設定については別梱のurl_scheme.pngをご参考ください。
